### Description

We created our game using pygame trinket, so we suggest that you upload our code there to try out the game. I will also have the images we used linked inside this github repo so that you can run the game with its designed appearence if you do so. 

Our game is an oregon trail type arcade game where you command a pirate ship across the ocean, experiencing different text-based events and making decisions that will help you survive the longest.
